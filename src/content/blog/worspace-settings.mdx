### How to install oh my posh

```sh
Set-ExecutionPolicy RemoteSigned
```

```sh
winget install JanDeDobbeleer.OhMyPosh -s winget

oh-my-posh init pwsh --config ~/jandedobbeleer.omp.json | Invoke-Expression

New-Item -Path $PROFILE -Type File -Force
```

After that it will create a file in the folder `C:\Users\usuario\Documents\WindowsPowerShell\Microsoft.PowerShell_profile.ps1`

Daí é so abrir este arquivo e colocar o tema que vc quer:

```
& ([ScriptBlock]::Create((oh-my-posh init pwsh --config "C:\Users\usuario\Documents\WindowsPowerShell\my-themes\the-unnamed.omp.json" --print) -join "`n"))
```

Depois instale a fonte para funcionar os icones com o terminal no modo administrador.

```sh
oh-my-posh font install

```
